Holly what..?

Holly Graph is a visualization tool which graphs directors ratings by movies. Each point on the graph displays the movie title, rating, and other data. We calculated linear regressions for each director. Based on each linear regression, it is possible ratings the top 5 Rising/Downward director. We also have the Most Awarded Directors/Actor and Unluckiest Directors/Actor

Data, rankings, numbers

We used data from the 250 directors better evaluated on votes by users of IMDb. Also, we use the list of films nominated in the category for best film at the Oscars between 2012 and 2015.
